# ATH9KFixup
An open source kernel extension providing patches for unsupported Atheros cards.

#

#

    AR946X (AR9462 & AR9463)
    AR9485
    AR9565

Features

Boot args:

    AR946X: (Default)
    AR9485: -ath9485
    AR9565: -ath9565
